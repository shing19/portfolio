# 利用 Python 学习一些小项目

大四在云科交换期间，修习“人工智能与机器学习”一课，接触到 python，逐渐感兴趣，开始用试用 python 尝试做些什么。

## 利用 Python + Flask 建站

本站即用 python + flask 搭建，通过 .edu 邮箱注册 Microsoft Azure 获取应用服务。本地写完后 git 传上云端。由于服务器设在东亚，不在国内，连接可能会有点慢。

参照 Miguel 搭建了类似微博的交流平台以学习，调用了 Microsoft 的翻译服务，尝试 Ajax，整个过程很难，因为没有基础，每一步都要学很多东西。
